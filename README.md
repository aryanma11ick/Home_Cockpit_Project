# Home Cockpit System

This project is a custom-built **home cockpit system** for flight simulation, focused on designing and developing modular cockpit panels using **PCB design, CAD modeling, and embedded systems**.

---

## ✈️ Overview

The goal of this project is to replicate real aircraft cockpit controls with high realism while keeping the system modular and scalable.

Currently, the project includes the development of:

- Overhead External Lights Panel  
- Annunciator (Alert) Panel  
- Custom Korry-style Switch Modules  

---

## 🧱 Project Structure
hardware/
├── pcb/ # KiCad design files + Gerbers
└── cad/ # STEP/STL files for panels and components

firmware/ # Microcontroller code (Arduino / ESP32)

docs/ # Documentation, diagrams, and images

## 🔧 Completed Modules

### 🟢 Overhead External Lights Panel
- Custom-designed panel for external lighting controls  
- CAD model for panel layout and mounting  
- Designed for integration with cockpit overhead section  

---

### 🟡 Annunciator (Alert) Panel
- Panel for status and warning indications  
- Uses custom-designed illuminated switches  
- Modular design for expansion  

---

### 🔘 Korry Switch Module (v2)
- Custom PCB designed in KiCad  
- Supports LED backlighting  
- Designed for reuse across multiple panels  
- Includes CAD models for mechanical integration
